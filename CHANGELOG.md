# 更新日志

本文件根据 Git 标签、提交历史和 README 中已有的更新记录整理。格式参考 Keep a Changelog，版本顺序为新到旧。

## 未发布

### 修复
- 修复 TXT 转世界书模块中源码和界面直接显示 Unicode 替换符的问题，改为明确的 `U+FFFD` 表达。
- 增强酒馆 API 调用兼容性：缓存 `generateRaw` 消息数组支持情况，避免反复提示“酒馆不支持消息数组格式”。
- 增强 AI JSON 响应解析：支持代码块提取、中文引号修复、尾逗号清理、括号补全、未转义引号修复和世界书结构兜底提取。
- 修复别名合并流程：兼容 AI 返回的不同字段名、布尔值表达、全局/局部配对编号和按顺序返回结果。
- 修复整理条目流程：支持清理 Markdown 代码块、JSON 包装响应和停止状态残留。

## v1.7.0 - 2026-04-25

### 新增
- 新增自动断点续传：处理期间定时保存状态，页面关闭前紧急保存，刷新或崩溃后可恢复。
- 新增 API 智能重试：扩展瞬态错误识别，酒馆 API 和自定义 API 均支持自动重试。
- 新增并行批次间世界书摘要同步，分批模式下将已有世界书摘要注入下一批次 prompt。
- 新增处理进度 ETA 预估，基于滑动平均计算并显示剩余时间。
- 新增批量操作前自动快照，替换、整理、合并等操作前保存世界书历史快照。
- 新增处理完成后自动去重建议，扫描疑似重复条目并提示使用别名合并。
- 新增 Prompt 模板可视化编辑，支持预览和编辑世界书、剧情、文风提示词模板。
- 新增增量导出功能，记录导出时间戳，支持只导出上次导出后的新增或修改条目。
- 新增响应过滤标签功能，支持黑名单移除 `thinking` 等标签内容。

### 修复
- 修复长篇小说处理必定失败的问题，用紧凑摘要替代完整世界书注入。
- 修复多处 XSS 注入漏洞，统一使用 HTML 转义处理动态内容。
- 修复四个高优先级功能缺陷。
- 修复数据完整性问题，包括 IndexedDB 竞态、事务完整性、合并去重和历史清理。
- 修复中等严重性问题，包括 ReDoS 防护、静默错误、文件限制、并发保护和死代码清理。

### 文档
- 优化帮助弹窗和提示信息，适配新增功能。
- 重写 README 和 TXT 转世界书模块文档。
- 更新版本至 v1.7.0，并修正 README 更新日志版本归属。
